# Legal translation inventory

The Dutch website currently has public privacy and cookie policies. It does not have a public Dutch terms and conditions page. `docs/sitemap-content-structuur.md` lists `/be/algemene-voorwaarden/` as recommended content that still needs to be added.

An English terms and conditions translation is therefore blocked and excluded from this localisation work. Legal terms must not be invented or translated from assumptions. Translation can begin only after a lawyer-approved Dutch source has been supplied and published as the factual authority.

The validated briefs in this directory cover only the existing privacy and cookie sources.
